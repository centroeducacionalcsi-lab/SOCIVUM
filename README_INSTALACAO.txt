SOCIVUM - Sprint 006 - Estrutura Organizacional

Inclui:
- Organização Matriz;
- Filiais com CNPJ próprio;
- Unidades vinculadas ao CNPJ da matriz ou filial;
- Espaços internos das unidades;
- Territórios de atuação;
- Seletor de contexto no topo: Organização > Filial > Unidade.

Instalação:
1. Extraia o ZIP.
2. Copie o conteúdo para C:\SOCIVUM.
3. Substitua os arquivos.
4. Rode:

npm run dev

Depois de testar:
git add .
git commit -m "Sprint 006 - Estrutura organizacional"
git push
