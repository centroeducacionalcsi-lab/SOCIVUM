# Sprint 008 - Cadastro Inteligente de Projetos

## Conceito
O projeto passa a ser um motor integrado do SOCIVUM.

Ao cadastrar um projeto, o sistema deverá preparar automaticamente:
- centro de custo;
- visão financeira;
- cronograma;
- metas;
- indicadores;
- oficinas/turmas previstas;
- ações/eventos;
- equipe;
- pasta documental;
- prestação de contas.

## Próximas evoluções
- Criar tabelas próprias para metas, cronograma e rubricas;
- Criar geração automática de centro de custo;
- Vincular beneficiários por turma;
- Integrar com financeiro e prestação de contas.
